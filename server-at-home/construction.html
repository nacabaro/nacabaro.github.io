Server at home

Server at home project

The server at home project was born as a way to share my tale on how I built my own server designed to serve the home. The idea for this came after a few friends at uni started asking me about how to build a home server like the one I am running. At first I was just gonna teach them how to make the server, but they prefered to have a list of things they could set up depending on their usecases, which was the thing that originated this website.

Building a Linux server from scratch also helped me a lot to learn about Linux in the first place, so I can say for myself that it is a great way to learn about Linux, plus, it's useful in the long run.
